# AI Чат

Чат с ИИ на основе G4F

# Преимущества

-   Быстрые ответы
-   Простота
-   Авторизация
-   И многое другое...

# Установка

```bash
# Linux/MacOS
python3 -m pip install -r requirements.txt

# Windows
py -3 -m pip install -r requirements.txt
```

Создать .env, назначить нужные переменные

```env
HCAPTCHA_SECRET_KEY=SECRET
HCAPTCHA_SITE_KEY=KEY
```

# Запуск

```bash
# Linux/MacOS
python3 run.py

# Windows
py -3 run.py
```
